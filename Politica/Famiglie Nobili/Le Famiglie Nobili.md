# Famiglie

Le famiglie nobili sono poche ma potentissime ed ognuna governa una delle Sei Stelle dell’Imperium.  
Ogni famiglia possiede:

- una guarnigione principale
- una flotta
- una branca scientifica
- un monopolio industriale
- un elder o una tecnologia esclusiva

Le famiglie combattono una guerra fredda costante fatta di:

- assassinii
- sabotaggi
- matrimoni politici
- guerre commerciali
- controllo delle miniere di Wylk
